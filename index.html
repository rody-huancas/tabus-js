View on GitHub
📡

BroadcastChannel Native

Uses the native browser API — no localStorage hacks, no polling, no server needed.

🔒

Type-safe Events

Generic Tabus gives you full TypeScript inference per event. No more any payloads.

🔄

Lifecycle Events

tab:join and tab:leave fire automatically when tabs open and close. No extra setup.

🛡️

Fallback & SSR Safe

Falls back to in-memory bus when BroadcastChannel is unavailable. Works in Next.js and Nuxt.

⚡

Throttle Built-in

Pass throttle option to limit emit frequency for high-rate events like mousemove or scroll.

🪶

Zero Dependencies

8.8 KB packed. No runtime dependencies. Tree-shakeable. CJS + ESM + types included.
